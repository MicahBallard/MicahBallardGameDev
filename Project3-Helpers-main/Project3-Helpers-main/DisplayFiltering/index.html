Display Filtering

We can build our own filtering solution fairly easily by using:

  • Custom classes (and using multiple classes where appropriate)
  • The "getElementsByClassName" function
  • Some simple JavaScript to toggle the inactive class of the elements with matching classes

The getElementsByClassName function is a little tricky, because it returns what it called an "HTML collection." However, you can see in our JavaScript code that it can easily be converted into an array.
