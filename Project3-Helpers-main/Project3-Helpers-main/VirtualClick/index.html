Virtual Click

We can make a button create a "click" event on another button. This can be useful for using the Bootstrap Tabbed Interface template with our own buttons instead of their tabs (so we still get the nice fade transition).

Use the buttons below to switch what tab content is loaded. Notice in the HTML that we're only using the tab-content part of the tabbed interface example since we don't need the actualy tabs themselves anymore to handle navigation.

These are our own custom buttons that we can use to drive the tabbed interface.

These are the tabs from the Bootstrap nav examples. If you'd prefer to use only your own buttons, you can hide these and the panel content transitions will still work.
